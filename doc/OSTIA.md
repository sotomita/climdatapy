# OSTIA

## Source
[Copernicus Marine Data Store](https://data.marine.copernicus.eu/product/SST_GLO_SST_L4_NRT_OBSERVATIONS_010_001/description)で公開されているテキストファイルをダウンロードし、NetCDF形式に変換する。

## ```download_kw```
指定する必要はない。

## ```.climdatarc```
以下の変数を設定する。

|変数名||
|---|---|
|```CopernicusMarineUsername```|Copernicus Marinのユーザ名|
|```CopernicusMarinePassward```|Copernicus Marinのパスワード|
